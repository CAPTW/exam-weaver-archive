# OCR language data

The `eng.traineddata` and `kor.traineddata` files are the fast models from
the official [tesseract-ocr/tessdata_fast](https://github.com/tesseract-ocr/tessdata_fast)
repository. They are distributed under the Apache License 2.0 and are bundled
so targeted recovery of small legacy answer cells is reproducible in packaged
builds.
